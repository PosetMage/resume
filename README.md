# Resume

* compile html to pdf:
  * ```docker compose run --rm compile```
* hold at local
  * ```docker compose up serve```
  * browse ```localhost:7777```
* For Github-Action
  * see ```.github/workflows```


pdf to png:
```
pdftoppm -png resume.pdf output
```
